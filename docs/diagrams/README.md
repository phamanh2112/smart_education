# System Diagrams

This directory contains architecture and design diagrams for the EduSmart platform.

| Diagram | File | Description |
|---------|------|-------------|
| Architecture | [`architecture.md`](./architecture.md) | System architecture (Nginx, Django, Celery, storage) |
| Context | [`context.md`](./context.md) | System context and external integrations |
| Database | [`database.md`](./database.md) | ERD schema with all tables and relationships |
| Class | [`class.md`](./class.md) | Python class diagram with methods and associations |

All diagrams use [Mermaid](https://mermaid.js.org/) syntax and render natively on GitHub.
